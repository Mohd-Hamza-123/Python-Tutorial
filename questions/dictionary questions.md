

10 Dictionary Questions

1. Create a dictionary person = {"name": "Amit", "age": 19} and print only the "name" value.


2. Given book = {"title": "Python", "price": 350}, update the "price" to 400.


3. Let mobile = {"brand": "Samsung"}. Add a new key "model" with value "M21".


4. Take city = {"name": "Mumbai", "state": "Maharashtra"}. Remove "state" using pop().


5. Given animal = {"type": "Dog", "legs": 4}, delete the key "legs" using del.


6. With laptop = {"brand": "HP", "ram": "8GB"}, access "ram" using the get() method.


7. Take food = {"item": "Pizza", "price": 250} and print all keys using keys().


8. Given bank = {"name": "SBI", "branch": "Pune"}, print all values using values().


9. Let game = {"name": "Cricket"}. Add {"players": 11, "type": "Outdoor"} using update().


10. For subject = {"name": "Math", "credits": 4}, remove the last key-value pair using popitem().
