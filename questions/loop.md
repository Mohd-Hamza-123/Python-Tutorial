

# Part 1: For Loop + Variables Only (10 Questions)

1. Write a for loop to print numbers from 1 to 10.
2. Print the square of every number from 1 to 20 using a for loop.
3. Print all even numbers from 2 to 50 using a for loop.
4. Print the cubes of numbers from 1 to 10.
5. Print numbers from 10 down to 1 (reverse loop).
6. Print the multiplication table of any number stored in a variable.
7. Print the sum of numbers from 1 to 100 using a for loop.
8. Print the factorial of a number using a for loop (no functions).
9. Print every second number from 5 to 50 (step 2 increment).
10. Ask the user for a number N and print numbers from 1 to N.
